# gowindow Landing Page

这是 gowindow macOS 应用的 landing page。

## 功能特点

- 简洁的技术风格设计
- 深色主题，符合开发者审美
- 响应式布局，支持移动端
- 平滑的动画效果

## 下载

前往 GitHub Releases 下载最新版本：  
[`https://github.com/immotal/gowindow/releases`](https://github.com/immotal/gowindow/releases)

## 文件结构

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 交互脚本
- `README.md` - 说明文档
