# gowindow Landing Page

这是 gowindow macOS 应用的 landing page。

## 功能特点

- 简洁的技术风格设计
- 深色主题，符合开发者审美
- 响应式布局，支持移动端
- 平滑的动画效果

## 使用方法

直接在浏览器中打开 `index.html` 文件即可查看。

或者使用本地服务器：

```bash
# Python 3
python -m http.server 8000

# Node.js (需要安装 http-server)
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 文件结构

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 交互脚本
- `README.md` - 说明文档
