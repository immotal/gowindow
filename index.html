$ 核心特性

⌨️

快捷键唤醒

在任何界面按下快捷键，立即唤醒窗口选择面板

🔤

字母导航

面板显示字母标签，输入对应字母快速定位窗口

🎯

二级定位

支持两级下钻：先定位到应用，再定位到具体窗口

$ 为什么选择 gowindow

> 相比传统窗口切换，支持两级下钻定位
> 键盘操作，无需鼠标，效率更高
> 字母导航，直观易记，学习成本低
> 轻量级工具，不占用系统资源

$ 使用示例

# 场景：你有多个 Chrome 窗口打开
1. 按下快捷键 ⌘ + Space
2. 输入 C 选择 Chrome
3. 如果 Chrome 有多个窗口，继续输入字母定位
4. 完成！窗口已切换
